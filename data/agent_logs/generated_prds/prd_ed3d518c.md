# 나만의 음악 재생 목록 관리 프로그램

## 1. 한 줄 정의
내 컴퓨터에 있는 음악 파일들을 자동으로 정리하고, 플레이리스트를 만들어 관리하는 CLI 프로그램

## 2. 문제 정의
음악을 좋아하는 사람들은 수백 개의 음악 파일을 가지고 있지만, 폴더에 무작위로 흩어져 있어 찾기 어렵고 체계적으로 관리하기 힘듭니다. 특히 아티스트별, 장르별로 정리하거나 기분에 따른 플레이리스트를 만들고 싶어도 수동으로 하기엔 너무 번거롭습니다. 이 프로젝트는 파이썬으로 음악 파일의 메타데이터를 읽고, 자동 분류하며, 나만의 플레이리스트를 관리할 수 있게 해줍니다.

## 3. 타겟 사용자
- 컴퓨터에 음악 파일(mp3, wav 등)을 가지고 있는 사람
- 음악을 체계적으로 정리하고 싶은 초보 개발자
- 간단한 CLI 도구를 만들어보고 싶은 파이썬 학습자

## 4. 핵심 기능
1. **음악 파일 스캔**: 특정 폴더를 스캔해서 모든 음악 파일 찾기
2. **메타데이터 추출**: 제목, 아티스트, 앨범, 재생시간 등 정보 읽기
3. **자동 정리**: 아티스트/앨범별로 폴더 구조 생성 및 파일 복사
4. **플레이리스트 생성**: 텍스트 기반으로 나만의 플레이리스트 만들고 저장하기
5. **통계 보기**: 내가 가진 음악 총 개수, 아티스트 수, 총 재생시간 확인

## 5. 기술 스택
- **언어**: Python 3.x
- **필수 라이브러리**:
  - `mutagen` (음악 파일 메타데이터 읽기, 무료)
  - `os`, `shutil` (파일/폴더 관리, 기본 내장)
  - `json` (플레이리스트 저장, 기본 내장)
  - `pathlib` (경로 처리, 기본 내장)
- **선택 라이브러리**:
  - `click` (CLI 인터페이스 개선용, 무료)
