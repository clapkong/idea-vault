# Spotify 플레이리스트 분석기

## 1. 한 줄 정의
Spotify 플레이리스트를 CSV로 내보내서 장르, 연도, 인기도를 분석하고 시각화 리포트를 생성하는 파이썬 프로그램

## 2. 문제 정의
Spotify 사용자는 자신의 음악 취향을 막연하게 느낄 뿐, 구체적으로 어떤 장르를 얼마나 듣는지, 어느 시대 음악을 선호하는지 데이터로 확인할 방법이 없습니다. 이 프로젝트는 플레이리스트 데이터를 분석해 자신의 음악 취향을 객관적으로 파악하고, 초보 개발자가 실생활에서 바로 쓸 수 있는 도구를 만들어 학습 동기를 높입니다.

## 3. 타겟 사용자
- Spotify를 사용하며 자신의 음악 취향을 데이터로 확인하고 싶은 사람
- 파이썬 기초를 배우고 음악과 연결된 실전 프로젝트를 만들고 싶은 개발 입문자
- API 없이 간단한 데이터 분석 경험을 쌓고 싶은 사람

## 4. 핵심 기능
- **CSV 파일 파싱**: Exportify로 내보낸 Spotify 플레이리스트 CSV를 pandas로 읽고 정제
- **장르 분포 분석**: 플레이리스트 내 장르별 곡 수를 집계하고 상위 5개 장르 추출
- **연도별 트렌드 분석**: 발매 연도를 기준으로 곡 분포를 시각화하여 선호 시대 파악
- **인기도 통계**: 각 곡의 인기도 점수 평균·최대·최소값 계산 및 분포 차트 생성
- **리포트 생성**: 분석 결과를 matplotlib 차트(막대그래프, 히스토그램)와 텍스트로 요약한 PNG/TXT 파일 출력

## 5. 기술 스택
- **언어**: Python 3.x
- **라이브러리**: pandas (CSV 파싱·데이터 처리), matplotlib (차트 생성)
- **데이터 소스**: Exportify (Spotify 플레이리스트 → CSV 내보내기 도구, 무료)
- **개발 환경**: VS Code 또는 Jupyter Notebook
- **버전 관리**: Git (선택사항)

## 6. 예상 일정
- **1주차 월-수**: pandas 기초 학습 (CSV 읽기, 데이터프레임 다루기, 필터링·그룹화) + Exportify로 샘플 CSV 내보내기
- **1주차 목-금**: matplotlib 기초 학습 (막대그래프, 히스토그램) + 간단한 차트 연습
- **1주차 주말**: CSV 파싱 코드 작성 및 장르·연도·인기도 컬럼 추출 검증
- **2주차 월-화**: 장르 분포 집계 함수 작성 및 막대그래프 생성
- **2주차 수-목**: 연도별 트렌드 히스토그램 작성 및 인기도 통계 계산 함수 구현
- **2주차 금**: 리포트 출력 기능 완성 (차트 PNG 저장, 텍스트 요약 TXT 저장)
- **2주차 주말**: 본인 플레이리스트로 테스트 및 README 작성

## 7. 주요 리스크
- **pandas/matplotlib 학습 시간 부족**: 공식 튜토리얼 대신 유튜브 "코딩애플 pandas", "나도코딩 matplotlib" 같은 2-3시간 압축 강의로 빠르게 필수 기능만 학습
- **Exportify CSV에 장르 누락 가능성**: 일부 곡은 장르 정보가 비어있을 수 있으므로 "Unknown" 카테고리로 처리하고, 데이터 정제 단계에서 결측치 확인 로직 추가
- **한글 인코딩 오류**: CSV 읽을 때 `encoding='utf-8'` 옵션 명시, 오류 시 `encoding='cp949'` 또는 `errors='ignore'` 사용
- **시각화 결과 해석 어려움**: 차트마다 제목·축 레이블·범례를 명확히 작성하고, 샘플 리포트를 먼저 만들어 출력 형식 검증
- **2주 내 완성 압박**: 1주차에 학습과 환경 설정을 완료하고, 2주차는 순수 구현에만 집중. 추가 기능(아티스트 분석 등)은 v2로 미루기
